Tutorial Steps

Add Bootstrap CSS in your HTML <head>:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

Basic card structure with image, title, and text:

<div class="card">
  <img src="image.jpg" class="card-img-top" alt="...">
  <div class="card-body">
    <h5 class="card-title">Card Title</h5>
    <p class="card-text">Card description...</p>
  </div>
</div>

Create responsive layouts with the grid system:

<div class="row g-4">
  <div class="col-12 col-md-6 col-lg-4">
    <!-- Card content -->
  </div>
</div>

Breakpoints: col-12 (mobile), col-md-6 (tablet), col-lg-4 (desktop)

Enhance cards with badges, buttons, and icons:

  • Badges: <span class="badge bg-primary">Tag</span>
  • Buttons: <a href="#" class="btn btn-primary">Click</a>
  • Icons: Include Bootstrap Icons CDN
  • Spacing: Use utility classes like mb-3, py-5
