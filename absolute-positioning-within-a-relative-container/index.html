It's all relative.

The Position Property

The position property lets us break our layout and nudge elements around in ways that we normally couldn’t. Instead of letting everything follow the normal document flow, we can yank them out of their spot and put them somewhere else.

So, how do we get started?

The Container

If you want to use absolute positioning, make sure that you have a relatively positioned container. This will act as a boundary for all of the absolutely positioned elements inside of it.

Taking Up Space

When you turn on absolute positioning, you are pulling an element out of the normal document flow. This means that it will no longer take up space within the layout. If the container that your element is in has no other content, the container will collapse.

In order to avoid container collapse and a jumble of elements stacking one on top of the other, make sure that you define the height and/or width of your container.

Aligning Your Elements

Finally, in order to nudge your absolutely positioned elements around, you will need to use one of the following 'helper' properties: top, right, bottom, left. The names of these properties correlate with each side of the relative container that your absolutely positioned elements are sitting inside of.

Next, assign a value. This value, like margins or padding, is how far away from that side of the container you’d like to place your absolutely positioned element. A positive value will push your element inwards; a negative value will pull it away from that side of the container.

Stacking Order

Don’t forget your z-index! If you would like to send an element to the front, remember to give it a higher z-index; if you would like to send an element to the back, give it a lower z-index.

By default, the z-index of each element is 0, with stacking order otherwise defined by the document flow.
